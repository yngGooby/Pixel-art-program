# Pixel art program
This started as the Odin Project's Etch-a-Sketch assignment, but I kept building and what you see here is v1.0 of my own pixel art app
built from scratch with vanilla JavaScript, HTML, and CSS, and no libraries, no frameworks.
id like to add this is v1.0 of the "Pixel art program"

## in v1.0 you have
- a dropper tool to select any color on the canvas
- a erases tool incase you make a slight mistake
- a clear canvas button incase you want to start a new!
- html's build in color picker tool so you can choose any color at your disposal
- a manual and sliding canvas changer so you can get an exact canvas size or a rough estamate!
- a mouse coordinate around the canvas incase you want specific
## what this project showcases for my skills so far are
- DOM manipulation — dynamically generating and updating grid cells in real time
- Event handling — coordinating multiple listeners to create smooth drawing behavior
- State management — tracking tool mode, draw state, and color across the app
- Problem solving — visible through my commit history as features were added incrementally
